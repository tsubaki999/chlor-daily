🎓 釣りの科学（読み物）

なぜクロロフィルが釣果と相関するの？
クロロフィル＝植物プランクトン（=餌の起点）。Chl-a 高濃度域では 動物プランクトン → 小魚 → 大型魚 と 食物連鎖 が活発化します。 特に Chl-a 0.3〜2 mg/m³ は程よい餌場として知られます。

なぜ潮目（SST フロント）に魚が集まるの？
水温が急変する境界（フロント）にはプランクトンが物理的に集積します。 そこに小魚が群れ、それを追って大型魚（ブリ・カツオ・マグロなど）が回遊します。 本アプリでは SST 勾配 0.3°C/km 以上 を強フロントとしてスコア加点しています。

SST と魚種の関係
冷水性（ブリ・タラ＝17℃前後）／中温（マダイ・ヒラマサ＝18-21℃）／暖水（マグロ・カツオ＝22℃〜）と 魚種ごとに好適水温帯が異なります。クロアは魚種別ガウシアンで適合度を評価。

📊 データソース
  • 🌡 海水温（SST）: Copernicus Marine OSTIA L4（解像度 1km、日次）
  • 🌿 クロロフィル（Chl-a）: Copernicus Marine BGC L4（OLCI 衛星由来）
  • 🌀 SST 勾配: OSTIA 4近傍差分から °C/km で算出
  • 🌊 水深: GEBCO 2024 グローバル（30秒解像度）
  • 🌬 風・波: Open-Meteo Marine API
  • 🌙 潮汐: JMA 調和定数（M2/S2/K1/O1）+ Schureman 簡易計算
